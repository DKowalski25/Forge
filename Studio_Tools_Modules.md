# Studio Tools

## Каталог возможных инструментов

> **Цель документа**
>
> Мы планируем разработать собственный набор инструментов для Unity
> (**Studio Tools**), который поможет автоматизировать рутинную работу
> всей команды.
>
> Ниже приведен список возможных модулей. Каждый из них решает
> определенную проблему, с которой разработчики сталкиваются ежедневно.
>
> Если вам не хватает какого-либо инструмента --- напишите его в чате

------------------------------------------------------------------------

# 1. Asset Import Pipeline

**Для кого**

-   Художники
-   Technical Artist
-   Программисты

## Проблема

Каждый раз после импорта нового изображения приходится вручную проверять
настройки.

## Что делает инструмент

-   автоматически выставляет PPU;
-   выбирает Filter Mode;
-   включает или отключает Compression;
-   выставляет Pivot;
-   определяет Sprite Mode;
-   распределяет спрайты по Sprite Atlas;
-   автоматически переименовывает файлы;
-   сортирует ассеты по папкам.

## Пример

**Сейчас:** художник импортирует десятки PNG и проверяет настройки
вручную.

**После внедрения:** достаточно скопировать изображения в проект ---
настройки применяются автоматически.

## Польза

-   Экономия времени
-   Единые настройки проекта
-   Минимум ошибок

------------------------------------------------------------------------

# 2. Character Builder

**Для кого**

-   Художники
-   Программисты

## Что делает инструмент

По папке со спрайтами автоматически создает:

-   Animation Clips
-   Animator Controller
-   Prefab
-   Collider
-   Sorting Group
-   необходимые компоненты

## Польза

Новый персонаж появляется в игре за несколько секунд.

------------------------------------------------------------------------

# 3. Prefab Builder

Создает готовые Prefab для окружения, предметов, NPC и интерактивных
объектов, включая массовую генерацию.

------------------------------------------------------------------------

# 4. Dialogue Toolkit

Позволяет просматривать диалоги, проверять портреты, эмоции, ID, ссылки
и ошибки без запуска игры.


------------------------------------------------------------------------

# 5. Tilemap Toolkit

Автоматизирует создание Tile Palette, Rule Tile, Collider, Scatter
Brush, Mirror Brush и проверку карт.


------------------------------------------------------------------------

# 6. Animation Toolkit

Автоматически создает Animation Clips, Animator Controller, Loop и
FPS-настройки по набору кадров.



------------------------------------------------------------------------

# 7. Validation Suite

Проверяет проект на:

-   Missing References
-   Missing Sprites
-   Broken Prefabs
-   Duplicate Assets
-   Naming Violations
-   Unused Assets


------------------------------------------------------------------------

# 8. Production Dashboard

Показывает статистику проекта:

-   ассеты
-   NPC
-   диалоги
-   анимации
-   память
-   ошибки
-   предупреждения


------------------------------------------------------------------------

# 9. Batch Toolkit

Позволяет массово:

-   переименовывать файлы;
-   менять Import Settings;
-   заменять спрайты;
-   создавать Prefab;
-   перемещать ассеты.

------------------------------------------------------------------------

# 10. Search Toolkit

Позволяет быстро находить:

-   использование ассетов;
-   зависимости;
-   неиспользуемые ресурсы;
-   ссылки между объектами.

------------------------------------------------------------------------

# 11. Environment Toolkit

Инструменты для быстрого создания окружения:

-   Scatter Tool
-   Random Rotation
-   Random Scale
-   Mirror Tool
-   Shadow Generator

------------------------------------------------------------------------

# 12. Audio Toolkit

Автоматизация импорта аудио, предпросмотр, поиск дубликатов и проверка
ссылок.

------------------------------------------------------------------------

На мой взгляд, если выбирать только самые полезные инструменты для первого релиза Studio Tools, то это будут:

1. Asset Import Pipeline — автоматизация импорта ассетов.
2. Validation Suite — поиск ошибок и битых ссылок.
3. Batch Toolkit — массовые операции.
4. Character Builder — создание готовых персонажей.
5. Dialogue Toolkit — если диалоги являются одной из основных механик игры.
